# malloc_test

For test your malloc :)
(For Epitech)

For use it, clone it in the malloc's folder and exec `./test.sh`

Option available:
```
--debug: Display the debugging options (errors / diff etc.).
--all: Test all.
--makefile: Test Makefile.
--basic: Basic test.
--advanced: Advanced test.
--bonus: Bonus test.
```
